Aim, Vision & Learning Outcomes

What this virtual lab is designed to achieve

🎯

Aim

To provide an interactive environment for students to understand, compute, and visualize maxima, minima, and saddle points of functions of two variables using the second derivative test.

🔭

Vision

Empower students to intuitively grasp multivariable calculus through step-by-step computation, symbolic math, and dynamic 3D surface visualization—bridging theory and practice.

📚

Learning Outcomes

  • Compute first-order partial derivatives fx and fy
  • Find critical points by solving fx = 0, fy = 0
  • Evaluate fxx, fyy, fxy
  • Calculate D = fxxfyy − (fxy)²
  • Classify: maximum, minimum, or saddle

Topic Introduction & Theory

Definitions, formulas, worked examples, and a quick-reference glossary

1. Functions of Two Variables

A function of two variables assigns a real number z to each ordered pair (x, y): z = f(x, y). Its graph is a surface in 3D space. We seek critical pointsPoints where both partial derivatives are zero — candidates for max, min, or saddle where the surface has a peak, valley, or pass.

2. First-Order Partial Derivatives

f_x = ∂f/∂x (treat y as constant) f_y = ∂f/∂y (treat x as constant)

A critical pointWhere f_x = 0 AND f_y = 0 simultaneously occurs where both fx = 0 and fy = 0.

3. Second-Order Partial Derivatives

f_xx = ∂²f/∂x² (double w.r.t. x) f_yy = ∂²f/∂y² (double w.r.t. y) f_xy = ∂²f/∂x∂y (mixed partial)

4. The Second Derivative Test — Discriminant D

D = f_xx · f_yy − (f_xy)²
ConditionClassification
D > 0 and f_xx < 0Local Maximum
D > 0 and f_xx > 0Local Minimum
D < 0Saddle Point
D = 0Inconclusive
Example 1 — f(x,y) = x² + y²
  • fx = 2x, fy = 2y → Critical point: (0, 0)
  • fxx = 2, fyy = 2, fxy = 0
  • D = (2)(2) − 0 = 4 > 0, fxx = 2 > 0 → Local Minimum
Example 2 — f(x,y) = x² − y²
  • fx = 2x, fy = −2y → Critical point: (0, 0)
  • D = (2)(−2) − 0 = −4 < 0 → Saddle Point
Example 3 — f(x,y) = −x² − y² + 4x + 6y
  • fx = −2x+4 = 0 → x = 2; fy = −2y+6 = 0 → y = 3
  • D = (−2)(−2) − 0 = 4 > 0, fxx = −2 < 0 → Local Maximum at (2, 3)
